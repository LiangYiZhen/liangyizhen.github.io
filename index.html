CodingPages申请SSL/TLS证书错误

参考链接：https://blog.csdn.net/lewky_liu/article/details/82432045

问题

因没有申请SSL/TLS证书，或者证书过期导致网站不能正常访问，即http与https的问题。

分析

由于我的个人站点是同时部署到GitHubPages和CodingPages上的，接着在阿里云域名解析里进行了配置，即默认的解析线路将我的域名指向 liangyizhen.coding.me，国外的解析路线则是指向了 liangyizhen.github.io。

之所以这样配置，是因为国内部分地区无法直接访问GitHub，自然就无法访问我部署在GitHub上的个人站点，于是我又选择了Coding.net的Pages服务，这样国内用户就可以快速访问到我部署在Coding Pages的个人站点，而国外用户则是快速访问到Coding Pages上的个人站点。

问题就出现在这里，因为我第一次申请SSL/TLS证书的时候，还没有解析境外的线路，所以很快就申请成功了。后来添加了国外线路的解析，这导致在Coding Pages的后台申请证书时无法通过验证，自然就申请失败了。

解决办法

由于我是在阿里云购买的域名，于是登陆到阿里云域名解析的后台系统，打开个人域名的解析设置，暂停对于境外线路的解析。这里暂停就行了，一般来说大概需要5分钟左右的生效时间，毕竟DNS解析是存在缓存的。

五分钟后，我又进入Coding Pages服务的后台，再一次申请SSL/TLS证书，果不其然，几秒钟后我申请证书成功。

最后，再次返回阿里云域名解析的后台，将境外解析的线路再次启用，嗯，完美。

我的考研方法论

我的考研方法论

一. 思想来源

  • 王道论坛

1.重点关注浙大板块以及中科大板块，请仔仔细细阅读每一篇好文章，尤其是跨考生，二本三本考上浙大的经历，因为他们的方法非常具有实战性。

2.如果你要报考的院校在王道论坛里能找见，请仔仔细细阅读里面的每一篇文章，走别人走过的路会少踩一些坑。

3.王道论坛的其他板块可以在闲暇之余阅读参考，只看985院校的板块，非985院校板块不必关注。

注：

无论你是不是计算机相关专业，都可以按照上述推荐使用王道论坛。因为，公共课是相同的，有很强的关联。如果你是计算机相关专业，请务必按照上述使用王道论坛。

  • 知乎

举个例子，如果你要考高数，英语，你可以在知乎中搜索：考研数学应该怎么学或考研英语怎么学。会出来一大堆的经验，选择攒数较高的文章进行阅读。

小结：

方法不是死的，你要把别人的经验结合你自己的实际情况来进行学习。

二. 我的情况

2.1 初试&复试概况

一志愿报考西工大软件，分数线370分，我的考研初试分数384分，由于大一接触过ACM，所以复试上机很轻松，不仅题目都做出来了，而且充分写了注释，感觉机试成绩分数应该不会太低。复试笔试只有两个选择题不会做，但是我用推理大致分析出了答案，我也不知道对错，其他都做的很顺，笔试应该不会太低。面试，我是第一个被面试的同学，有点小激动和刺激，但不紧张，老师问的问题偏向概率论和机器学习，我在面试表现应该普普通通吧，亮点是本科获得过国家级励志奖学金，每学期均可获得校级优秀学生奖学金，党员，三好，本科做过自然语言处理的项目，辅修英语双学位，英语面时六位老师一致说我的英语发音好听，估计也是亮点吧，所以我的面试分数应该普普通通吧。

2.2 复习情况

可以量化的数据：恋恋有词反反复复背了37遍，英语真题反反复复过了4遍，英语阅读反反复复过了5遍，高数18讲反反复复看了5遍，线性代数反反复复过了4遍不止，数学真题做了4遍，专业课复习遍数4遍，政治背诵反反复复无法计数。

当然还有很多无法量化的数据。

三. 考研全程大规划

这是我综合了王道论坛北大，浙大，中科大学长们的经验得出的共性

3.1 高数

记住！！！9月1日，是一个至关重要的日期。

要想真正上考场，高数真题至少刷四遍，一共32套真题，试想一天做一套的话，需要一个月的时间。那么做四遍真题需要的是9，10，11，12月，正好四个月，如果晚于9月1日这个日期才开始动真题的话，高数真题肯定做不完的。

因此，请在9月1日前把那些18讲，基础强化视频课，笔记，1000题，1800题，660题等等凡是你用过的完完全全搞懂了！！！

3.1.1 高数用书

高数两位名师：汤家凤，张宇，无他！

（市面上所有的老师网课我多多少少都听过，由此得出的结论）

注：如果你考研起步时间是前一年的11月或者12月，强烈推荐把张宇和汤家凤的网课都听了，包括基础班和强化班。

理由：这两位老师的课程是完美的互补的，有些东西张宇讲清楚了，汤家凤没讲清楚，有些东西汤家凤讲清楚了，张宇没讲清楚（比如微积分的物理与实际应用部分，张宇讲的不清楚，汤家凤讲的清清爽爽，还有很多地方是汤家凤讲的不清楚，张宇讲的简洁明了，总之，谁用谁知道。）

用书：你听谁的网课用谁的书籍，不要混搭。

张宇：1000题+18讲

汤家凤：1800题+高数辅导讲义

真题：张宇真题大全解（只买这一本真题，必须！！！）

押题卷：李林，张宇等等（我没做押题卷，只把真题做透了，好像李林又押中了？？？）

其他：660题（学有余力）

3.1.2 高数全程规划

我认为，理想的高数开始复习时间是前一年的11月，因为这样可以把张宇和汤家凤的网课都看掉。

如果你的起步时间是前一年的11月，请这样来：

前一年11月-3月：把前一年的汤家凤基础强化视频看完并且一定！一定！一定！要记笔记。

3月-8月31日：全程跟上当年张宇的基础强化班视频，并且不断消化吸收你记得汤家凤张宇的笔记。

如果你的起步时间是当年3月，请这样来：

3月-8月31日：全程跟上当年的张宇的基础强化班视频，并且不断消化吸收张宇的笔记。

9月-10月：真题一遍

10月-11月：真题二遍

11月-12月：真题三遍+模拟题

12月： 真题四遍+模拟题

我为何推荐后期跟张宇：因为他的团队比较完整，这就比汤家凤一个人要做得好，就好比1000题有专门的高昆仑来讲，而汤老师一个人毕竟精力有限。

3.1.3 高数学习核心

  • 笔记要不断反反复复理解消化，不能前期记过后后期就扔掉了。
    反反复复理解笔记的目的是让你能在脑海中形成高数的知识体系，
    如果你在9月1日前不能合上书心里映射出高数知识框架，包括每章的重点，考点等等
    那我认为，你的高数复习是不合格的。

  • 1000题或者1800题做三遍：
    第一遍：一道一道题认认真真在A4纸上做。
    凡是 你做错的 和 你认为的难题 和 你认为的好题 都标记出来
    同时，还要标记出 你认为完全掌握 的题目。

    第二遍：一道一道题认认真真在A4纸上做。
    不过，这一遍不要做你在第一遍中标记出来的 你认为完全掌握 的题目
    你在第一遍做题中会因为粗心做错一些题，第二遍时这些题目肯定能做对，
    所以，在第一遍你标记出 你做错的 题目中把这些粗心做错的题目删去。
    这样，你通过两遍刷题，已经淘汰了一大堆对你无意义的题目了。

    第三遍：只做你标记出的并且第二遍筛掉后的 你做错的 和 你认为的难题 和 你认为的好题
    反反复复把这些题搞懂

  • 学有余力：18讲一字一字的理解消化吸收
    18讲给我的感受是，它是一本提高你内功的书，我看了5遍，反反复复一字不落。
    最终做真题时，我会分析了，拿到题目不会慌。当然，这是学有余力，如果时间不够的话，
    就挑着看吧。

9月1日前的所有高数的学习，都是为了这个阶段做铺垫，前期你复习不合格，这个阶段你是很痛苦的！！！

真题第一遍：

  • 时间：9月1日-9月31日（严格遵守）

  • 卡时间：2小时30分必须做完（一定要按这个时间）
  • 最好还是估分，自己把主观证明题的分值扣分扣得多一点
  • 题目标记：凡是 你做错的 和 你认为的难题 和 你认为的好题 都标记出来
              同时，还要标记出 你认为完全掌握 的题目。

真题第二遍：

  • 时间： 10月1日-10月25日（严格遵守）（按题型做，你做题的速度会提升，肯定用不了一个月）
  • 按题型做，一道一道认认真真把过程写在A4纸上
  • 把第一遍中你因为粗心做错的题目如果你做对了就把他从你的标记册中删掉，
    题目标记：凡是 你做错的 和 你认为的难题 和 你认为的好题 都标记出来
              同时，还要标记出 你认为完全掌握 的题目。

真题第三遍：

  • 时间：10月26日-11月20日（此时，你做题速度会更快）
  • 按题型做，一道一道认认真真把过程写在A4纸上
  • 把第一遍中你因为粗心做错的题目如果你做对了就把他从你的标记册中删掉，
    题目标记：凡是 你做错的 和 你认为的难题 和 你认为的好题 都标记出来
              同时，还要标记出 你认为完全掌握 的题目。
              重点关注你在前两遍中标记出来的题目。

真题第四遍：

  • 时间：自主安排
  • 只做你在前三遍中标记出来的并筛掉的题目，你做错的 和 你认为的难题 和 你认为的好题，完完全全搞懂它们

3.4 英语

英语主要分这么几大块，独立攻破：语法，单词，阅读，翻译，完型，新题型，作文

3.4.2 单词

（1）单词用书

考英语一的同学：恋恋有词

考英语二的同学：恋恋有词+老蒋带你8天搞定英二真题词汇

（2）单词学习的核心方法

两字：重复

大家都知道艾宾浩斯曲线，要求我们严格按照固定时间节点去背诵复习单词，然而每个人的时间记忆规律是不一样的，所以我们吸取艾宾浩斯的精华，不必严格按照他的时间节点去记忆，他的思想的精华是重复，所以我们这样来背单词：

  1. 每天记单词的时间：早晨饭前，中午饭后，下午饭后，半夜入睡前

  2. 频率：前几遍你可以背的慢一点，比如一天一个unit，或者两个unit，但是一定要不断重复你前一天背过的单词，比如你早上+中午背完一个unit，那么晚上和半夜要及时回顾复习已经背过的unit，你前一天背了unit1，那么第二天不仅要背unit2，还要复习unit1，依次类推。

注意：前几遍背单词可以放慢速度，但是要求慢慢加速，比如你前几遍一天背1-2个unit，但是越往后频率要越快，我10月后的最高频率是60分钟5个unit，相当迅速了，感觉背单词时嘴在高速的运动，根本不用经过大脑的思考，一切都是本能反应。

  1. 背单词的内容：我们思考一下，考研英语需要我们写单词吗？不需要，即使是作文，那都是模板（必须套模板，但是这个模板不是一般的模板，得是专属你的模板，我一会会叙述。），我们只需要在阅读中能认出单词的意思就好，所以我们背单词只背：恋恋有词上的中文释义，不背例句。

前几遍你可以背1-2个主要释义，但是越往后你就应该逐渐扩充释义，最后把整本书的释义都背下来。

注意：单词千千万万不要看任何视频，时间守恒定律，前文有叙述。

注意: 单词背诵要天天背，直到考试前一天你才可以停止背诵。

3.4.4 翻译

（3）书籍：他的视频课配套书籍即可。

（1）学习时间： 11月

（2）老师推荐： 刘晓艳或者新东方宋逸轩（英语二听刘晓艳的即可）

（3）书籍： 配套讲义即可，不用买书

（1）学习时间： 10-11月

（2）老师推荐：英语一：新东方唐迟；英语二：谭剑波（强烈推荐）

3.4.7 作文

（1）学习时间：9-10月

注意：

作文不用像王江涛那样背那么多，把时间留给专业课和数学。

3.4.8 阅读

考研英语：得阅读者得天下(极其重要！！！！)

（1）学习时间：贯穿全程始终，每天都要碰阅读

（2）老师推荐：英语一：唐迟或者何凯文；英语二：谭剑波（强烈推荐，必看！！！）

（3）书籍：这里只推荐英语二：老蒋阅读十二式
说明：谭剑波的英语二阅读思路和老蒋阅读十二式的思路完美一致，不冲突，谁用谁知道。

（4）阅读复习的核心：

其实，考研英语真题可以不按套卷做，每种题型有规定的时间来做完，做题时间和顺序：

作文（大，小）：30-40min —>阅读：20min/篇 —>新题型 ；15min —>翻译：20min
—>完型 ：15min

所以，平常练习时，你可以不按套卷做，但是请严格按照这个时间标准做完每种规定的题型。

阅读做四遍，第一遍要四篇一起做，把时间控制在20*4=80min内，

第二遍，按题型来做（阅读12式那本书），体会每种题型的阅读思路。

第三遍，按题型来做，同上，目的是强化题型思路。

第四遍，按套卷做，把握试卷阅读整体结构。

不要担心记住答案的问题，我反正记不住答案。

3.4.9 真题的选择

英语一：黄皮书

英语二：黄皮书或者老蒋真题

黄皮书排版比较好，老蒋真题讲解出色。

注：为何11，12月极其重要？

因为，所有押题，背诵核心都在这两个月出啊，我也很无奈。。。。。

3.5.2 政治复习安排

  • 8月-10月：全程跟徐涛的网课，不看基础班，直接强化班，同时，做肖秀荣1000题（最好过3遍）。

  • 11月-12月： 狂背！！！，具体安排如下文。

3.5.3 得11月，12月者，得考研政治！！！！（极其重要）

考研政治，主要分为两大板块，咱们来一一攻破。

  • 选择题：把市面上所有的押题卷都买到手，只做选择并且搞懂。

  • 简答题：只背两个：肖4+徐涛20题（原题都在里面，就看你背不背了）
    两个都背，徐涛会教你如果答案不在资料里该怎么应对。
    总之，都背的不少，加油吧。

3.6.2 专业课资料获取

首先，你得搞到你报考目标院校的历年期末试题还有历年考研真题，如果实在搞不到就算了，有的同学从淘宝买，这种渠道非常不好，我一开始打算报考北京大学软微院，然后买了相关的真题，我看内容全是复制粘贴一些可有可无的网络内容，资料意义不大。

比价靠谱的获取资料途径：

  • 打开qq，找群，输入你报考的目标院校的目标群，比如你要考西工大计算机学院，就输入：西工大2020计院考研（如果没有，则换一种表述：2020西工大计算机考研），加群后看群文件查找相关资料。
  • 进入王道相应的板块获取资料。

注：如果你今年打算报考西工大软微院，请联系我。

3.6.3 专业课复习安排

计算机专业的同学，建议报考专业课考2-3门的院校，这是为什么呢？

因为：如果只考一门专业课，你可以试想一下如果一个跨考的同学和你在一起考试，你学了四年计算机，而跨考的同学学习计算机的时间小于四年，那同样都是复习仅仅一门专业课，科班出身的你是不是比较不占优势。

如果你考四门专业课，别忘了你还有数学，那么时间精力可能会太累了。

所以，理想的专业课是考2-3门。

注： 确定院校后，首先你应该获取到目标院校专业课去年以及前年的考试大纲，并对照着历年期末真题和考研真题，自己分析分析考纲有没有重点和高频考点，总之，就是运用你的分析能力解析考纲。考西工大软微院考生联系我。

基础不好的同学，建议从4月或者5月开始，基础好的同学，建议从6月开始，专业课一共过四遍，1.5个月过一遍：

第一遍：6，7月

第二遍：8，9月

第三遍：10月

第四遍：11月

注：专业课408考生请务必3月开始复习专业课，如果晚于5月，除非你基础特别好，否则不建议报考408院校。

什么才可以称为“一遍”：

王道天勤书本一个字一个字的看并且习题都做了+历年期末题做完+历年考研真题做完

具体的知识点都在书里叙述，我这里不在赘述，但请注意，数据结构的复习天勤上出现的基础代码一定得自己会写，然后思考如何设计代码。考试时多写注释。

  • 首先，我不推荐往北京上海考，原因：

  • 2019年，我觉的是985计算机考研的里程碑。因为这一年比之前有很大的变化：

985院校复试更看重考生能力，即使你400+也照样刷。
985初试越来越难，复试机试越来越难，这是我的感受，今年西工大专业课比16，17，18难的多了。

所以，如果你本科编程不好，没有论文和项目，成绩一般般，不太建议你报考985。

4.1 张宇汤家凤不会讲到的一些知识点

4.1.1 微分方程

微分方程现在张宇汤家凤主要讲解的是：按类求解，对号入座。
然而，缺点就是计算量大，计算耗时长。
我要补充的知识点是：
微分算子法
优点是：解题速度极快，提供了又一种解决微分方程的方法体系，可以用来验证最终结果。
我将在bilibili直播中对微分算子法进行详细的阐述。

4.1.2 无穷级数

推荐看看毛纲源老师的数学复习全书无穷级数部分，很有用！

4.1.3 不定积分

陈文灯表格法，张宇也有叙述。

4.1.4 多维随机分布

方浩老师的卷积公式计算起来很方便。

4.2 关于择校

4.3 如何看分数线

分数线有四条：
国家线，校线，院线，录取平均分数线。

你要重点关注录取平均分数线，如果你有信心比这个分数线高，那么就可以报考。
